The dataset copied in this directory obtained from the Mulan multi-target data repository (http://mulan.sourceforge.net/).

scene.mat
Original source: M.R. Boutell, J. Luo, X. Shen, and C.M. Brown. Learning multi-label scene classification. Pattern Recognition, 37(9):1757-1771, 2004.

yeast.mat
Original source: A. Elisseeff and J. Weston. A kernel method for multi-labelled classification. In Advances in Neural Information Processing Systems 14, 2001.
